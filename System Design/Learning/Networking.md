#SystemDesignConcepts
Networking as about connecting devices and allowing them to communicate
3 key layers that come up in system design interviews:
1) Network Layer: (IP, InfiniBand) = Handles routing
2) Transport Layer (TCP, UDP, QUIC) = Provide reliability, ordering, and flow control
3) Application Layer (REST, Websockets, DNS, WebRTC) = Provides abstractions for different types of data
